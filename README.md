# 🚀 DSA Visualizer Hub

An interactive web application designed to visualize **Data Structures and Algorithms (DSA)** through real-time animations, helping students and developers understand complex concepts in an intuitive way.

---

## 📌 Overview

**DSA Visualizer Hub** is a comprehensive educational platform that demonstrates the working of various Data Structures and Algorithms with animated visualizations, step-by-step execution, complexity analysis, and reference code.

Whether you're a beginner learning DSA or preparing for coding interviews, this project provides an engaging way to understand how algorithms and data structures work internally.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/dsa-visualizer-hub.git
```

Navigate to the project

```bash
cd dsa-visualizer-hub
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:5173
```
